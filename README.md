# rx-onchange-subscriber

## Install

Add the following in your root build.gradle
```
allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

Add the following to your project dependency
```
dependencies {
		compile 'com.github.cgcgbcbc:rx-onchange-subscriber:0.2.0'
}
```
